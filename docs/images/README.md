# Images / media for the project README

Put screenshots / photos here and reference them from the root `README.md`
with a relative path, e.g. `![Hexapod in Gazebo](docs/images/hexapod-gazebo.png)`.

**Video** (YouTube): GitHub Markdown can't embed a video player, so use a
clickable thumbnail that links to the video:

```markdown
[![Watch the demo](docs/images/video-thumbnail.png)](https://youtu.be/YOUR_VIDEO_ID)
```
